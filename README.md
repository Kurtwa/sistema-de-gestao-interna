# Sistema de Gestao Interna

Prototipo web local do **RIS Control**, um sistema de gestao operacional para academias.

O app centraliza:

- RIS - Registro de Incidencia e Solucao.
- Tarefas com responsavel, prazo e status.
- Checklists operacionais.
- Painel da gerencia.
- Historico de movimentacoes.
- Perfis simulados de acesso.

## Como abrir

Abra `index.html` ou `ris-control.html` no navegador.
